Kanban Board
This project is a Kanban board application with JWT-based authentication. The application allows users to securely log in, manage their work tasks, and organize them into different columns based on their status.

Features:
Secure login page
JSON Web Token (JWT) authentication for secure API access
Token stored in localStorage for subsequent authenticated requests
Session expiration handling
Ability to log out and invalidate the session
Getting Started:
To run the project, you will need to:

Set up a .env file for the server with the following:

A username for the database.
A password for the database.
A secret key for JWT authentication.
Implement the required authentication methods in both the server and client code.
